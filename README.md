# 📘 CourseHub

📝 Description

CourseHub provide premium courses, exclusive content, and personalized learning paths to accelerate your growth
and stay ahead in the modern world.

🛠️ Tech Stack

This project uses the following technologies:

⚡ Next.js – React framework for building fast and scalable web apps
🎨 Tailwind CSS – Utility-first CSS framework for rapid UI development
🌼 DaisyUI – Tailwind CSS component library for beautiful UI
🔐 BetterAuth – Authentication system for secure user management
🎬 Animate.css – Library for adding smooth CSS animations
🎯 Lucide React Icons.

✨ Features
🔐 User Authentication (Login / Signup)
🎨 Clean and responsive UI design
⚡ Fast performance with server-side rendering (SSR)
📱 Fully responsive across devices
🎬 Smooth animations using Animate.css
🧩 Reusable UI components with DaisyUI
